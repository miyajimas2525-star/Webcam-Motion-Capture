Webcam-Motion-Capture
Webカメラの映像から、Google MediaPipeを用いてリアルタイムでアバター（VRM）を動かすモーションキャプチャシステムです。

主な機能と工夫した点
1. 数学的なアプローチによる関節角度の算出
MediaPipeから取得した25箇所以上の座標データ（Point Annotation）を元に、C#のMathf.Atan2を用いて、2点間のベクトルから回転角度を動的に計算しています。

メリット:ライブラリに頼り切るのではなく、3D空間上の回転を自力で制御しているため、アバターのモデルに合わせた柔軟な調整が可能です。

2. Quaternion.Lerpによる動きの平滑化
カメラ映像から取得した生データには特有のノイズ（ガクつき）がありますが、Quaternion.Lerp を用いた線形補間（係数0.1f）を実装することで、実用的なレベルまで滑らかな動きを実現しました。

3. Tスタンス補正（オフセット機能）
アバターの初期姿勢と、実際のユーザーの姿勢のズレを解消するため、左右それぞれの腕に独自のオフセット値（右腕：-140°、左腕：-20°など）を設定できる仕組みを構築しました。
